Пояснительная записка к игре "Lost"

Игра представляет собой RPG c 4 локациями, где вы должны помочь главному герою найти выход из другой вселенной.
Прокачивайте своего персонажа и выберетесь из чужого мира.

Для игры используются 5 файлов с кодом:
	main.py ; windows_image.py ; pers.py ; items.py ; shells.py
Также используется 1 папка с изображениями:
	data

Чтобы запустить игру нужно запустить файл main.py
При запуске открывается стартовое окно с 3 кнопками:
 	Кнопка Играть - начать игру
	Кнопка Инструкция - открывает инструкцию игры
	Кнопка Авторы - показывает авторов игры
При нажатии на кнопку играть вы сможете прочитать сюжет игры и после чего управлять персонажем.
Герой передвигается на wasd(стрелочки).
